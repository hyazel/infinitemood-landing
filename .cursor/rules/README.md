# .cursor/rules

Ce dossier contient les fichiers de règles Cursor pour chaque module du projet.

- Créez un fichier par module (ex: `audio.md`, `ui.md`, etc).
- Décrivez dans chaque fichier les conventions, règles de style, ou guidelines spécifiques à ce module.

Exemple de structure :

- .cursor/rules/audio.md
- .cursor/rules/ui.md
- .cursor/rules/fmod.md
